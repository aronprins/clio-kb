# Clio KB Overview

Clio KB is a static knowledge base starter that renders Markdown from `docs/` inside a single lightweight site shell.

It is designed for teams that want:

- plain Markdown files in the repository
- a clean landing page and sidebar
- full-text search without a framework
- straightforward static hosting

## How it works

The app loads two things:

1. `site/content.json` for navigation metadata.
2. Markdown files from `docs/` for page content.

That means the authoring model stays simple. If you can edit Markdown and JSON, you can shape the whole site.
